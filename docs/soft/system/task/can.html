CAN通信制御タスク¶

CAN通信を制御するタスクってなにやってんの

目的¶

OS上の様々なタスクとマイコン外との通信の仲介をする。
本タスクの役割は以下の二つ

  • 他のタスクが外部へメッセージを送信する際に、OS提供の内部メッセージを送信するだけでいいようにする。

  • 他のタスクが外部からのメッセージを受信する際に、OS提供の内部メッセージを受信するだけでいいようにする。

つまりは翻訳である。
