Driver(ドライバ)¶

組み込みのシステムには OS や Task(タスク) 以外にDriverが存在する。
ここではそのDriverについて説明する。

概要¶

組み込み開発ではマイコン毎に制御方式が大きく異なる。
Driver(ドライバ) はその制御方式の差分を吸収し、 OS や Task(タスク) に変わらないIFを提供する。
つまり、違うマイコンにシステムを載せ替える際、最も変更量が激しいのがこの Driver(ドライバ) である。
