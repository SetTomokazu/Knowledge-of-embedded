IDE(統合開発環境)¶

マイコン依存の開発環境
対象のマイコンに対してのビルドやデバッグ機能をGUI込みで提供する。
当然マイコン依存の為開発するマイコンが変われば別のIDEに乗り換える必要がある。
そして、組み込み開発向けIDEの主な目的は

  • ビルド

  • 実機への焼き込み

  • 実機デバッグ

であり、そこにコーディングは含まれていない。
なんならテキストエディタとしてさえも使いにくい代物になっている。
その為、コーディングには専ら他の テキストエディタ を使う。
