システム構造¶

組み込みで使用されているプログラムの概略を示す。

組み込みのマイコンは電源が入っている状態で常に通信、検知、演算、描画といった制御を行っている。
この制御をリアルタイムで動作させ続けるために、また相互に動作させるために何をしているのかを解説する。
大体仕事ではある程度出来上がったシステムを改修する為に全体を把握する必要があり、
その一助になれば幸いである。
