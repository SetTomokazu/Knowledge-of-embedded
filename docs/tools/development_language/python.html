長所¶

  • 参考サイトが豊富

  • ファイル入出力が楽

  • 文字列解析が楽

  • 追加でモジュールを入れることでグラフ作成などより色々なことができる。

  • スクリプト言語の為、ソースファイルと実行ファイルが同一

短所¶

  • 実行環境が無いと動かない

  • 引き継ぎってなんでしょうか

  • 追加のモジュールもダウンロードとインストールが出来ないと元も子もない。
