測定機器¶

組み込み開発とはつまり基盤上のマイコンを制御することであり、
動作の成否を確認する際に一番信用が置けるものは基盤上に流れる制御信号そのものである。
その制御信号は、開発しているマイコン上ではレジスタで判断し内部で処理された後の結果しか確認できない。
その為、専用の測定機器を使用し実際の波形を見る必要がある。
代表的な測定機器は以下の通りである。
