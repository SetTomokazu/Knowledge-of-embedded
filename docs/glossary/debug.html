Debug¶

デバッグ。
組み込み開発はデバッグをするのも一苦労である。
というのも外部割り込みや通信を完全に再現できるシミュレータなんてほぼないのである。
その為、実際の基盤を使用し、動作を確認する。
これを実機Debugとかいう。
